# CLAUDE.md

## Project goal 
This project is a romantic web quiz for Alba.
It is meant to be played together on Saturday.
The purpose is to remember shared moments, laugh, feel tenderness, and strengthen the relationship.

## Core product principles
- This is not a generic quiz app.
- The emotional experience matters as much as functionality.
- The UI must feel warm, elegant, intimate, and memorable.
- The experience should feel personal and crafted.
- Prefer emotional polish over feature bloat.

## Visual direction
- Warm and premium aesthetic
- Soft colors, not saturated
- Clean typography with elegant hierarchy
- Spacious layouts
- Rounded but refined surfaces
- Subtle shadows
- Smooth animations
- Mobile-first

## Avoid
- Generic SaaS/dashboard UI
- Harsh contrast or neon colors
- Childish or cartoonish styling
- Excessive clutter
- Overly cheesy romantic design
- Loud effects
- Dense text blocks

## UX rules
- One main action per screen
- Strong visual hierarchy
- Clear progress through the quiz
- Clear answer states
- Results screen should feel rewarding and emotional
- Keep copy concise, warm, and natural

## Engineering rules
- Reuse components
- Reuse design tokens
- Avoid random one-off styles
- Keep structure maintainable
- Prefer a small set of polished screens over many mediocre ones

## Workflow
When asked to improve UI:
1. Audit current screen
2. Identify concrete visual problems
3. Propose design direction briefly
4. Implement
5. Self-review and refine
